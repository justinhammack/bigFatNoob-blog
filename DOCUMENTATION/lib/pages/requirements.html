Requirements

Server requirements

  • WordPress 3.2 or higher
  • MySQL 5 or higher
  • PHP 5 or higher
  • cURL library support (compiled into to PHP)
  • GD 2 library support

We've tested our themes to ensure they are compatible with all the popular web browsers like Google Chrome, Opera, Firefox, Safari, Internet Explorer 7+! They should also all be web standards compliant.
