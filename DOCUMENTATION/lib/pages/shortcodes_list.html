Images

Easy image
Parameters:
  • 'src' - Image URL.
  • 'size' - Image size (xl, l, m, s, xs).
  • 'title' - Image title.
  • 'group' - Lightbox group. Images can be grouped in galleries.
  • 'link' - External link. If the link exists lightbox effect is automatically disabled.
  • 'crop' - The code allows you to align the cropping region to different edges of the image.
    • 'c' - Position in the center (this is the default).
    • 't' - Align top.
    • 'tr' - Align top right.
    • 'tl' - Align top left.
    • 'b' - Align bottom.
    • 'br' - Align bottom right.
    • 'bl' - Align bottom left.
    • 'l' - Align left.
    • 'r' - Align right.
Default usage:
[easy_image src="#" size="s" title="Image Title" group="" link="" crop="c"]
Custom image
Parameters:
  • 'src' - Image URL.
  • 'width' - Custom image width.
  • 'height' - Custom image height.
  • 'title' - Image title.
  • 'lightbox' - Use lightbox effect.
  • 'group' - Lightbox group. Images can be grouped in galleries.
  • 'hover' - When the mouse cursor is on the image, the image changes its transparency. The value of '0 'disables this effect.
  • 'image_alignment' - You can align images according to the text around it, using the following alignments:
    • 'left' - Aligns the image in the left margin and wraps the text that follows the image.
    • 'right' - Aligns the image in the right margin and wraps the text that precedes the image.
    • 'center' - Aligns the center of the image with the surrounding text.
  • 'link' - External link. If the link exists lightbox effect is automatically disabled.
  • 'target' - Shows the link in a new window. If the parameter 'target' is set to 1 and the parameter 'link' is not empty, when you click on the image, the link will show in a new window.
  • 'crop' - The code allows you to align the cropping region to different edges of the image.
    • 'c' - Position in the center (this is the default).
    • 't' - Align top.
    • 'tr' - Align top right.
    • 'tl' - Align top left.
    • 'b' - Align bottom.
    • 'br' - Align bottom right.
    • 'bl' - Align bottom left.
    • 'l' - Align left.
    • 'r' - Align right.
Default usage:
[custom_image src="#" width="200" height="200" crop="c" title="Custom image" link="" lightbox="lightbox" group="test_group" hover="1" target="0" image_alignment="left"]
