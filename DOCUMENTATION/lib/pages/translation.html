Translation

Translating theme into another language

  1. Download and install: http://www.poedit.net/
  2. Go to the languages folder of the theme. and open the en_EN.po file in the languages folder. /languages/en_EN.po
  3. Open POedit and translate the English language into your preferred language.

    When complete, you’ll want to save the file twice, as two separate files – a .po file and a .mo file. When you save the files, you must name them according to your language code.

    Find a list of language codes at http://codex.wordpress.org/WordPress_in_Your_Language. As an example, the language code for English is en_EN, so you would save the translated files as en_EN.po first, then en_EN.mo. When that is done, simply upload the files your theme folder using FTP program.

  4. Edit wp-config.php file

    Finally, if you haven’t done so already, you’ll need to make sure your wp-config.php file matches your language files. For example, if you’re using a French translation, you’ll need to add the fr_FR.po and fr_FR.mo files to your theme folder, then set your language in wp-config.php, like this:

    define('WPLANG', 'fr_FR');
  5. Save your wp-config.php file, upload it to your WordPress installation, and you are ready to go.

Change default theme texts

If you use the program Poedit and if you set the default language of English en_EN, then you can change all default texts without editing a template codes. This is very convenient because if the new upgrade will change the code, you will need to change the texts again.

Then you set your language code in the file wp-config.php, like this (for English language):

define('WPLANG', 'en_EN');

Your wp-config.php file is loacated in the main directory where all your core WordPress files are located.
